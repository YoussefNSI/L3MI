__construct()

Digraph.php : 35

Construit le digraphe G à partir de son nombre de noeuds $nrNodes et du tableau $arcs codant son ensemble d'arcs et sa fonction f.

public __construct(string $name, int $nrNodes, array<string|int, array<string|int, mixed>> $arcs) : mixed

$arcs doit être indexé sur [[E]] et associer chaque clé e dans [[E]] (un identifiant d'arc) à un tableau indexé sur 0..1 qui encode la paire ordonnée de noeuds f(e) :

  • l'élément d'index 0 encode la queue de e dans la plage [[V]],
  • l'élément d'index 1 code la tête de e dans la plage [[V]].

$arcs doit être trié lexicographiquement dans l'ordre croissant des identifiants d'arc et des nœuds, c'est-à-dire,

  • il est trié dans l'ordre croissant des clés (identifiants d'arcs),
  • pour deux arcs e_i et e_j tels que f(e_i)=(x_i,y_i), f(e_j)=(x_j,y_j) :
    • e_i<e_j implique (x_i<x_j ou (x_i=x_j et y_i<=y_j)).
Parameters
$name : string

Le nom de G.

$nrNodes : int

Le nombre de noeuds.

$arcs : array<string|int, array<string|int, mixed>>

L'encodage du tableau f.
